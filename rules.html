Game Rules

  1. Goal:
    • Build four foundation piles, one for each suit, from Ace to King.
  2. Tableau Setup:
    • Seven piles of cards, starting with one card in the first pile and adding one more card to each pile until you have seven cards in the last pile.
    • Only the top card of each pile is face-up.
  3. Move Cards:
    • Move cards between piles to create descending sequences (King, Queen, Jack, etc.) and alternating colors (red and black).
    • You can only move a King into an empty space.
  4. Foundation:
    • Start each foundation pile with an Ace and build up in order (2, 3, 4, etc.) by suit.
  5. Stockpile:
    • Draw cards from the stockpile (remaining cards) when you can't make a move.
  6. Winning:
    • Win by moving all cards to the foundation piles in order, from Ace to King.
  7. Losing:
    • If you can't make any more moves and the stockpile is empty, the game is over.
